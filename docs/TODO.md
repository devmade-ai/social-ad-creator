Plan: (1) bump `react` + `react-dom`; (2) bump `@vitejs/plugin-react` 4.x → 6.x in the same step (the latest plugin matches React 19 properly); (3) re-run all 164 tests and the manual TESTING_GUIDE.md scenarios, paying attention to the BottomSheet TDZ test, the modal `<dialog>` sync, and the PWA install/update flow; (4) check console for new Strict Mode warnings about `useEffect` ordering.
